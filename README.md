
# vehicular mobility model VMM

## to execute 
Install the dependecies

`pip install simpy`
`pip install matplotlib `

Then RUN the simulate.py
`python3 simulate.py`

it will generate the data in the form of json files

`RSU.json`
`vfs.json`
`user_vehicle.json`
`distances_rsu.json`
`distances_vfs.json`
`vehicles.json`

## Algorithms used
1. KMM
2. GMDC
